# Київ — Взаємодопомога

Легка платформа оголошень для жителів Києва: «Потрібна допомога» / «Можу допомогти», фільтри по районах, приховані контакти та P2P-реквізити.

## Стек

- **Frontend:** Next.js 15, React 19, Tailwind CSS
- **Backend:** Supabase (PostgreSQL + API routes на Vercel)

## Швидкий старт

```bash
npm install
cp .env.example .env.local
# Заповніть ключі Supabase, виконайте supabase/schema.sql
npm run dev
```

Без Supabase стрічка порожня; публікація потребує підключеної бази (див. `DEPLOY.md`).

Детальна схема БД: [DATABASE.md](./DATABASE.md)  
Деплой за 60 хв: [DEPLOY.md](./DEPLOY.md)

## Безпека

- Контакти та картки — лише після кліку «Показати контакти»
- Капча + ліміт 3 оголошення/год
- Скарги з авто-приховуванням після 5 репортів
- Попереджувальний банер про шахрайство
